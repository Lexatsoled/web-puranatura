# Calendario Tentativo (8 semanas)

- Semana 1: Fase 0 completa (inventario, secretos, CSP report-only, backup prueba). Gates básicos.
- Semana 2-3: Fase 1 (hardening, authZ, CSP lista, IA segura, SAST/DAST). Gate seguridad.
- Semana 3-4: Fase 2 (OpenAPI completa, migraciones, catálogo paginado/cache, analytics endpoint). Gate contratos.
- Semana 4-5: Fase 3 (UX/A11y/perf: lazy catálogo/modales, imágenes, LHCI, axe). Gate a11y/perf.
- Semana 5-6: Fase 4 (observabilidad, tracing, alertas, CI/CD con canary). Gate observabilidad + CI/CD.
- Semana 6-7: Fase 5 (refactor SOLID/Clean, complejidad, pre-commit obligatorio, ADRs). Gate maintainability.
- Semana 8: DR drill y hardening supply-chain (SBOM/licencias/attestation), revisión final de SLOs y cierre de deudas.
