# Instrucciones de Plan y Ejecución

Esta carpeta contiene los manuales paso a paso para cada agente/IA involucrado en la corrección de los hallazgos críticos de Pureza-Naturalis-V3. Cada archivo incluye contexto, dependencias, comandos y criterios de validación. Sigue el orden por fase y respeta los checkpoints antes de avanzar.

## Archivos
- `01_Codex_Primary.md`: orquestación general y validaciones cruzadas.
- `02_GPT-4.1_Seguridad_y_Backend.md`: tareas de seguridad crítica y lógica compleja backend/frontend.
- `03_GPT-5-Mini_Optimización_UI.md`: mejoras ligeras, encoding y service worker.
- `04_Grok_Code_Fast_Scripts_Tests.md`: automatización de scripts, pruebas y SEO dinámico.

Actualiza estos documentos si el alcance cambia para evitar pérdida de contexto entre turnos o agentes.
