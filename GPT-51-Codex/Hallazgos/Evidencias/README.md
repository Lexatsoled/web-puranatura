# Evidencias visuales

---

version: 1.0  
updated: 2025-11-19  
owner: QA Visual

Coloca en esta carpeta las capturas exportadas desde GitHub (por ejemplo `ci-fails-2025-11-19.png`, `checks-summary.png`). Al adjuntarlas:

1. Nombra el archivo siguiendo `YYYYMMDD-contexto.png`.
2. Actualiza [`Hallazgos/log-debug.md`](../../Hallazgos/log-debug.md) para enlazar la imagen con sintaxis Markdown: `![CI Fails](Evidencias/20251119-ci-fails.png)`.
3. Indica en el text log el commit/PR al que pertenece la evidencia.

> Nota: Las capturas incluidas en el PR #2 se referencian actualmente mediante el enlace público del repositorio. Cuando dispongas de los binarios, cópialos aquí para cerrar el requisito.
